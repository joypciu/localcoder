ชุดโครงการชุมชนที่สร้างขึ้นบน LocalCoder

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ LocalCoder ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-localcoder](https://github.com/awesome-localcoder/awesome-localcoder) และ [localcoder.cafe](https://localcoder.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน

---
